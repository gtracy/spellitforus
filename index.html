Spellitfor.us

Spellitforus is an automated spelling test system that uses the telephone to deliver the words.

If you press a telephone, you'll be called and asked to spell the list of words in that test. You "spell" them using the telephone keypad. You can take the tests available to left or you can create your own.

Note that when using the keypad for your answers, you only need to press a key once per letter. For example, the letter 'b' is 2 (not 22 as it would be when typing a text message).

For example...

  • "bad" -> "223"
  • "banana" -> "226262"
  • "car" -> "227"
